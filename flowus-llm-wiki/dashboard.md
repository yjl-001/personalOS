# 00 Dashboard

这是个人知识库 OS 的首页。它应该是工作台，不是装饰页。

## 今日入口

- 快速记录：新想法、新问题、新资料先进入 `Raw Sources` 或 `Questions`。
- 待处理资料：嵌入 `Raw Sources / 未处理` 视图。
- 正在探索的问题：嵌入 `Questions / 正在探索` 视图。
- 最近更新知识页：嵌入 `Wiki Pages / 近期更新` 视图。

## 知识地图

嵌入 `Wiki Pages` 的几个视图：

- 概念地图
- 主题页
- 自我知识
- 待验证
- 稳定知识

## 当前问题

嵌入 `Questions`：

- 高优先级
- 待沉淀
- 已回答

## 输出管道

嵌入 `Outputs`：

- 草稿
- 已完成
- 按类型

## 最近日志

嵌入 `Log / 最近日志`，显示最近 10 条。

## 系统维护

固定链接：

- `99 System / LLM 维护规则`
- `Raw Sources`
- `Wiki Pages`
- `Questions`
- `Outputs`
- `Log`

## 每周操作

每周至少做一次：

```text
请通过 FlowUS MCP 对我的知识库做一次 weekly review：
1. 处理最近未入库的高价值资料
2. 检查待沉淀问题
3. 找出待验证知识页
4. 给出下周最值得研究的 3 个问题
5. 写入 Log
```
